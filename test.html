四校数学成绩分析系统测试

功能说明

本系统已成功修改以支持四校数学成绩分析，主要功能包括：

  • ✅ 读取包含学校、考号、班级、姓名、数学、校内名次、联考名次的Excel文件
  • ✅ 支持灵活配置分析参数（学科、满分、及格线、优秀线等）
  • ✅ 针对九年级数学120分满分进行优化分析
  • ✅ 自动解析数据并生成各校各班统计
  • ✅ 生成学校排名、班级排名等分析报告
  • ✅ 支持导出Excel格式的分析报告
  • ✅ 支持导出PDF格式的分析报告
  • ✅ 提供可视化图表展示

使用方法

  1. 启动开发服务器：npm run dev
  2. 在浏览器中打开 http://localhost:3000
  3. 点击"分析配置"标签页，选择"九年级数学"预设配置
  4. 点击"文件上传"标签页，上传 example/四校数学.xls 文件
  5. 查看"数据分析"页面的统计结果
  6. 查看"成绩表格"页面的详细数据
  7. 使用右上角的"导出报告"按钮导出Excel或PDF

分析配置说明

系统支持灵活的分析配置，针对不同学科和年级：

  • 预设配置：九年级数学（120分满分，72分及格，108分优秀）
  • 自定义配置：可设置学科名称、年级、满分、及格线、优秀线
  • 实时调整：修改配置后自动重新分析数据
  • 智能分布：根据配置自动调整分数分布区间

数据格式要求

Excel文件应包含以下列（第一行为表头）：

  • 学校：学校名称
  • 考号：学生考号
  • 班级：班级名称
  • 姓名：学生姓名
  • 数学：数学成绩
  • 校内名次：校内排名
  • 联考名次：联考排名

分析报告内容

系统将生成包含以下内容的分析报告：

  • 总体统计：总学生数、总学校数、总班级数、整体平均分、及格率、优秀率
  • 学校统计：各学校排名、学生人数、班级数、平均分、及格率、优秀率
  • 班级统计：各班级总排名、校内排名、平均分、及格率、优秀率
  • 学生成绩表：包含联考排名、校内排名、学校、班级、姓名、考号、成绩等详细信息
  • 可视化图表：各学校对比图、各班级对比图、成绩分布图等

导出格式

  • Excel格式：包含多个工作表的完整分析报告
  • PDF格式：包含统计表格和图表的PDF报告
